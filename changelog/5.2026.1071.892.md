Bug fixes for reports, Stock Assembly, MyInvois, and general improvements. (*892*)

<!-- truncate -->

## Updates

- **Stock**:
  - Set default CashFlowType for clipboard imports when Excel values are empty (#7691).

- **MyInvois (E-Invoice MY)**:
  - MyInvois import now requires access rights for creating or editing customers/suppliers.

## Bug Fixes

- **Reports**:
  - Fixed E-Invoice 1 field mapping for Amount with Tax.
  - Fixed Profit & Loss with Cash Flow (Amt Exclude Tax) to use Supplier Code for Purchase documents to get correct Tax Amount, and exclude CN with date greater than Date To.

- **General**:
  - Fixed deleting documents to prompt Acceptable Transaction Date approval dialog, including Draft & Cancelled documents (#7699).
  - Fixed "Edit As New" not triggering OnNew and DIY events.

- **Stock**:
  - Fixed View > Stock Available Balance > Back Order Detail and Job Order detail items to hide when master item has been fully transferred (#7673).
  - Fixed Stock Assembly causing system exit when changing BOM Template (#7694).

- **MyInvois (E-Invoice MY)**:
  - Fixed auto map to only apply to active tax codes.
  - Fixed non-English locale causing conversion error when submitting consolidated documents (#7693).
